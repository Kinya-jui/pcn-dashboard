## Tool 2 XLSForm Modifications — pulldata() Auto-fill
## =====================================================================
## Add these rows to the Tool 2 (pcn_tool.xlsx) survey sheet.
## Each row goes ABOVE the existing question it auto-fills.
## The enumerator selects the PCN → these fields populate automatically.
##
## COLUMN VALUES TO SET:
##   type        = calculate
##   name        = (same as the Tool 2 question name)
##   calculation = pulldata('pcn_lookup', 'FIELD_NAME', 'pcn_key', ${facility})
##
## Then on the ORIGINAL question (integer/decimal type), change:
##   default     = ${FIELD_NAME}        ← this pre-fills it with the calculated value
##   relevant    = false()              ← hides it (enumerator can't change it)
##   OR leave visible so enumerator can verify/correct if needed
## =====================================================================
